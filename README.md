# Trailer Hunt

**A Movie trailer showcasing Web application**

## Features

- **Firebase** for user authentication
- **TMDB API's** for fetching real movies data
- **Custom hooks** for Handling API's call
- **A.I Intergration (Gemini)** so user can ask to A.I about the movies
- **Redux and Redux toolkit** for state management

## Tech Stack

- **React:** for Building U.I
- **Redux and Redux toolkit:** for managing state globally
- **Firebase:** for handling user authentication efficiently
- **React router dom:**  for handling routing
- **Tailwind CSS:**: for build an U.I beautiful and responsive
- **Gemini Integration:** for adding A.I chatbot to the app.
